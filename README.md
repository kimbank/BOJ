# BOJ Format

## File Name

[문제 번호] . [확장자]

_ex)_ _`12345.cpp`, `01000.py`_

<br/>
<br/>

## C/C++

```C++
// # [문제 번호]번   ex) 12345
// # [문제 제목]   ex) Hello World!

```

<br/>
<br/>

## Python

```python
# [문제 번호]번    ex) 12345
# [문제 제목]    ex) Hello World!

```
